# bot/src/idl/

This folder holds the generated Anchor IDL (Interface Definition Language) file
for the Delta Vault on-chain program.

## How to generate it

1. Install Anchor CLI:
   npm install -g @coral-xyz/anchor-cli

2. Build the program:
   cd ../../programs/delta_vault
   anchor build

3. Copy the generated IDL here:
   cp ../../target/idl/delta_vault.json ./delta_vault.json

4. The bot will automatically detect and load it on next start.

## What the IDL does

The IDL tells @coral-xyz/anchor exactly how to call each instruction
in the on-chain program — account layouts, argument types, error codes.

Without it, anchorClient.ts falls back to "stub mode" (logs only, no txns).
With it, every updateNav / updateRiskOracle / updateStrategy call actually
hits the on-chain program.
